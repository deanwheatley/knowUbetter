## Components

### Profile Header
- Username
- Rank and special badges
- Member since date
- Back navigation

### Sub-Navigation Tabs
- Overview (stats, streaks, categories, leaderboard positions)
- Badges (all earned badges sorted by rarity)
- Props History (all props received with messages)
- Activity (recent actions and achievements)

### Overview Tab
- Total stats (kudos, accuracy, rank)
- Current and longest streaks
- Questions by category with progress bars
- Leaderboard positions across all boards

### Badges Tab
- Grouped by rarity (Legendary, Epic, Rare, Common)
- Badge name, icon, description
- Earned date
- Dynamic badges highlighted

### Props History Tab
- Chronological list of props received
- Sender name
- Prop type and kudos amount
- Message text
- Timestamp
- Pagination

### Activity Tab
- Chronological feed of all actions
- Question answers (correct/incorrect)
- Props sent/received
- Badges earned
- Streaks achieved
- Questions submitted/approved
- Pagination
