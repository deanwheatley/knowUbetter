- **Primary Blue**: `#3B82F6` - Main actions, buttons
- **Success Green**: `#10B981` - Correct answers, achievements  
- **Warning Orange**: `#F59E0B` - Notifications, pending items
- **Error Red**: `#EF4444` - Wrong answers, warnings

## Background & UI
- **Background**: Light gray `#F9FAFB`
- **Cards**: White `#FFFFFF` with subtle shadows

- **Hover**: Slightly darker shade of base color
- **Active**: Even darker with subtle glow effect
- **Disabled**: 50% opacity with gray tint

- **Success Flash**: Bright green with fade
- **Level Up**: Gold particles with sparkle effect
- **Streak Fire**: Orange to red gradient
- **Achievement Pop**: Rainbow gradient burst
