## Components

### Weekly Allowance Display
- Shows remaining props of each type
- Kudos value for each prop type
- Countdown to weekly reset

### Recipient Selection
- Search/filter users
- List of users with rank and kudos
- Single selection (radio button style)

### Prop Type Selection
- Visual cards for each prop type
- Shows remaining count
- Selected state highlight
- Disabled if none remaining

### Message Input
- Required text area
- Minimum 10 characters
- Character count display

### Send Button
- Disabled until all fields valid
- Confirmation animation on success

### Success State
- Celebration message
- Summary of what was sent
- Updated allowance display
- Options to send another or return to dashboard
