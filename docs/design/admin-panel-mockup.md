## Components

### Sub-Navigation
- Approval Queue (pending questions count)
- Configuration (all system settings)
- User Management (search, view, manage users)
- Analytics (system stats)

### Approval Queue
- List of pending questions
- Full question preview
- Duplicate detection warning
- Edit capability
- Approve/Reject actions
- Rejection reason modal

### Configuration Panel
- Quiz settings (kudos, limits, timing)
- Props settings (allowances, rollover)
- Streak rewards
- Weekly reset settings
- Save changes button

### User Management
- Search/filter users
- User list with stats
- Pagination
- User detail view
- Admin actions (reset, promote, ban, delete)

### Analytics (Future)
- Engagement metrics
- Question difficulty distribution
- Category balance
- User growth
- Props activity
